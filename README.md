# QQ群聊 - OpenClaw Agents

一个基于OpenClaw Agents的QQ群聊模拟器，可以在网页端模拟多Agent群聊场景。

## 功能特性

### 群聊功能
- 从OpenClaw配置中读取Agents列表
- 支持邀请/移出Agent到群聊
- 多Agent群聊互动
- 用户消息触发Agent响应
- 新成员加入自动欢迎（需点击开始按钮触发）
- 实时WebSocket通信

### 模仿游戏 🎭
- Agent轮流模仿指定对象谈论特定话题
- 每轮表演后，其他Agent根据表演内容评分（0-10分）
- 评分参考表演内容，确保公正评分
- 实时显示评分进度和分数更新
- 每轮淘汰得分最低的Agent
- 最终产生冠亚季军
- 支持查看每位选手的平均得分

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动服务

```bash
npm start
```

服务启动后访问 http://localhost:18800

## 使用说明

### 群聊模式
1. **邀请Agent**: 在右侧面板的"可用机器人"中找到Agent，点击"邀请"按钮
2. **开始聊天**: 拉入新Agent后，点击"🎤 开始聊天"按钮触发欢迎
3. **发送消息**: 在输入框中发送消息，群内的Agent会依次响应
4. **移出Agent**: 点击已加入Agent旁边的"✕"按钮移出群聊

### 模仿游戏模式
1. **切换到模仿游戏**: 点击顶部的"🎭 模仿游戏"标签
2. **设置游戏参数**:
   - 输入模仿对象（如：特朗普、关羽、马斯克）
   - 输入话题（如：如何拯救地球、AI的未来）
3. **开始游戏**: 点击"开始一轮"按钮
4. **观看表演**: Agent会轮流进行模仿表演
5. **评分环节**: 其他Agent会根据表演内容打分
6. **淘汰环节**: 每轮得分最低的Agent被淘汰
7. **查看结果**: 游戏结束时显示冠亚季军排名和平均得分

## 项目结构

- `package.json` - 项目配置
